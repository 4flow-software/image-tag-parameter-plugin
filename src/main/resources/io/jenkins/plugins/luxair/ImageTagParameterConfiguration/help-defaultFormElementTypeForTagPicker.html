This setting enables administrators to define the default HTML element type used for selecting Docker image tags.

SELECT ... Utilizes an HTML <select> element, restricting input to selection from a dropdown menu without the option for free keyboard input.
AUTOCOMPLETE ... An HTML <input> element paired with an additional <datalist>. Unlike SELECT, this option permits users to freely enter text while benefiting from autocomplete assistance.
AUTOCOMPLETE_STRICT ... Similar to AUTOCOMPLETE, with the added behavior that, upon a blur event, if its value is not among the available tags, it resets itself to a defaultTag.
